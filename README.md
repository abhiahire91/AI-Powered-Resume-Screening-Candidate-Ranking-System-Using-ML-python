<h2 style="color:#16a34a;">📌 Project Overview</h2>

<div style="background:linear-gradient(135deg,#022c22,#065f46); padding:18px; border-radius:12px; color:white;">
This project is an <b>AI-powered Resume Screening System</b> that automatically evaluates and ranks resumes based on their similarity to a given job description using <b>NLP techniques</b>.

<br><br>

It helps recruiters:
<br>
✔️ Automate candidate shortlisting  
✔️ Reduce manual effort  
✔️ Improve hiring efficiency  
</div>

<h2 style="color:#16a34a;">🚀 Key Features</h2>

<ul>

<h2 style="color:#16a34a;">🧠 How It Works</h2>

<ol>
  <li>📂 Load dataset of resumes</li>
  <li>🧹 Clean text using NLP preprocessing</li>
  <li>📊 Convert text into TF-IDF vectors</li>
  <li>📏 Compute similarity using cosine similarity</li>
  <li>🏆 Rank candidates based on score</li>
</ol>
