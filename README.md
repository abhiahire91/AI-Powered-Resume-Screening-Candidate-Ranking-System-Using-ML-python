AI-Powered Resume Screening & Candidate Ranking System

📌 Overview
This project is an NLP-based system that automatically analyzes and ranks resumes based on their similarity to a given job description. It helps streamline the candidate screening process using machine learning techniques.

⚙️ Tech Stack
Python
NLP (Natural Language Processing)
Scikit-learn
Pandas
Matplotlib & Seaborn

🚀 Features
Resume text preprocessing (cleaning, stopword removal)
TF-IDF vectorization for feature extraction
Cosine similarity for resume-job matching
Candidate ranking with match scores
Data visualization (bar charts, histogram, pie chart, heatmap, wordcloud)
Function to evaluate new resumes

📊 Output
Top & Bottom candidate comparison
Resume score distribution
Candidate match classification (High / Low)
Keyword visualization from job description

▶️ How to Run
pip install nltk scikit-learn wordcloud seaborn
python your_script_name.py

💡 Use Case
Automates manual resume screening and helps recruiters identify the best candidates efficiently.
