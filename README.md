<h2 style="color:#16a34a;">📌 Project Overview</h2>

<div style="background:linear-gradient(135deg,#022c22,#065f46); padding:18px; border-radius:12px; color:white;">

This project is an <b>AI-powered Resume Screening System</b> designed to automate the process of evaluating and ranking job candidates.

<br><br>

Instead of manually reviewing hundreds of resumes, this system uses <b>Natural Language Processing (NLP)</b> techniques to compare resumes with a given <b>Job Description (JD)</b> and assign a similarity score.

<br><br>

The higher the score, the better the candidate matches the job requirements.

<br><br>

💡 <b>Goal:</b> Reduce manual effort, improve hiring efficiency, and enable data-driven recruitment.

</div>

<hr>

<h2 style="color:#16a34a;">⚙️ Tech Stack</h2>

<ul>
  <li><b>Programming:</b> Python</li>
  <li><b>Libraries:</b> Pandas, NumPy, Scikit-learn, NLTK</li>
  <li><b>NLP Techniques:</b> Text Cleaning, Stopword Removal</li>
  <li><b>Feature Extraction:</b> TF-IDF Vectorization</li>
  <li><b>Similarity Measure:</b> Cosine Similarity</li>
  <li><b>Visualization:</b> Matplotlib, Seaborn, WordCloud</li>
</ul>

<hr>

<h2 style="color:#16a34a;">🧠 Project Workflow (Step-by-Step)</h2>

<ol>
  <li><b>📂 Data Loading:</b> Load resume dataset containing candidate details</li>
  <li><b>🧹 Text Preprocessing:</b>
    <ul>
      <li>Convert text to lowercase</li>
      <li>Remove special characters using regex</li>
      <li>Remove stopwords using NLTK</li>
    </ul>
  </li>

  <li><b>📊 Feature Engineering:</b>
    <ul>
      <li>Convert resumes & job description into numerical vectors using <b>TF-IDF</b></li>
    </ul>
  </li>

  <li><b>📏 Similarity Calculation:</b>
    <ul>
      <li>Use <b>Cosine Similarity</b> to measure how closely each resume matches the job description</li>
    </ul>
  </li>

  <li><b>🏆 Ranking System:</b>
    <ul>
      <li>Assign scores to each resume</li>
      <li>Sort candidates based on highest match score</li>
    </ul>
  </li>
</ol>
